### `textInput` {#textInput}

- Type: {<br />
  &nbsp;&nbsp;&nbsp;&nbsp; `styles?:` {[`text?: CustomStyle`](./sharedTypes/#customStyle), [`container?: CustomStyle`](./sharedTypes/#customStyle)}, <br />
  &nbsp;&nbsp;&nbsp;&nbsp; `placeholder?:` {`text?: string`, [`style?: CustomStyle`](./sharedTypes/#customStyle)}, <br />
  &nbsp;&nbsp;&nbsp;&nbsp; `characterLimit?: number`, <br />
  &nbsp;&nbsp;&nbsp;&nbsp; `disabled?: boolean` <br />
  }

Custom details for the chat's text input. <br />
`styles` object defines the styling for the `text` element and its box `container`. <br />
`placeholder` object's `text` property is the initial text that is displayed on the input before the user makes any actions and
`style` can be used to customize it. <br />
`characterLimit` is the maximum number of charactes that can be inserted into the text input. <br />
`disabled` is used to prevent the user from inserting text. This also comes with its own custom styling. <br />

#### Generic example

<ComponentContainer>
  <AiAssistantBrowser
    textInput={{
      styles: {text: {fontSize: '16px'}, container: {maxHeight: '50px', backgroundColor: '#dffeff'}},
      placeholderText: 'Insert text here...',
      characterLimit: 10,
    }}
    service={{custom: {demo: true}}}
  ></AiAssistantBrowser>
</ComponentContainer>

<LineBreak></LineBreak>

#### Disabled example

<ComponentContainer>
  <AiAssistantBrowser
    textInput={{disabled: true, placeholderText: 'Text not allowed...'}}
    service={{custom: {demo: true}}}
  ></AiAssistantBrowser>
</ComponentContainer>

### `dragAndDrop` {#dragAndDrop}

- Type: `boolean` | [`CustomStyle`](./sharedTypes/#customStyle)
- Default: _false_

Configuration for the drag and drop functionality. If the [`service`](HERE) you are using facilitates file uploads - this is defaulted to _true_. <br />
Additionally once the user drops a file, it will only be accepted if the [`service`](HERE) is configured to accept it. For example if it only
allows images through the use of [`camera`](HERE), a dropped _.txt_ file will not be accepted.

#### Example

<ComponentContainer>
  <AiAssistantBrowser
    dragAndDrop={{backgroundColor: '#80ff704d', border: '5px dashed #52c360'}}
    service={{custom: {demo: true, mixedFiles: true}}}
  ></AiAssistantBrowser>
</ComponentContainer>
