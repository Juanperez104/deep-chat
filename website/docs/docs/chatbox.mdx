# Chatbox

Properties related to table container.

### `chatboxStyle` {#chatboxStyle}

- Type: [`CustomStyle`](./sharedTypes#CustomStyle)

Object containing custom CSS styling properties for the chatbox element. <br />

#### Example

import ActiveTable from '@site/src/components/table/activeTableBrowser';
import AiAssistant from '@site/src/components/table/aiAssistantBrowser';
import TableContainer from '@site/src/components/table/tableContainer';
import LineBreak from '@site/src/components/markdown/lineBreak';
import BrowserOnly from '@docusaurus/BrowserOnly';
import TabItem from '@theme/TabItem';
import Tabs from '@theme/Tabs';

:::
:::note
`minWidth`, `height`, `minHeight`, `maxHeight` properties are not supported.
:::
