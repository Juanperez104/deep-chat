# Sticky

Example to help you stick the component to any side of the screen - furthering a chat-like experience to your users. The core method to achieve this is to
set the [`containerStyle`](../../docs/styles#containerStyle) property with _position: "fixed"_ and defining the side you want to component to sit on e.g. _right: "7%"_.

<h3>Example code</h3>

```html
<!-- This example is for Vanilla JS and should be tailored to your framework (see Frameworks) -->

<deep-chat
  containerStyle='{
    "borderRadius": "10px",
    "position": "fixed",
    "bottom": "0px",
    "right": "7%",
    "zIndex": "1"
  }'
  initialMessages='[
    {"text": "Hey, how are you?", "role": "user"},
    {"text": "I am doing great, how about you?", "role": "ai"},
    {"text": "What is the meaning of life?", "role": "user"},
    {"text": "This completely depends on the person.", "role": "ai"}
  ]'
  directConnection='{"demo": true}'
  stream="true"
></deep-chat>
```
