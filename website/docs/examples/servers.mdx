The following are links to full backend example projects that work with Deep Chat. Each one includes a preconfigured set of endpoints for
handling text messages, file attachments and streamed responses; with variations for a `basic` server that you can
customize to fit your use-cases and an `OpenAI` server that can act as a proxy for connecting to its API.

The projects are preconfigured to work with the example [UI project](https://github.com/OvidijusParsiunas/deep-chat/tree/main/example-servers/ui).
Once the server and UI are both are running - they should be able to communicate with each other right out of the box.
